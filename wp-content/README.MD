
### 2. Install the Plugin

* Copy `edumate-lms/` folder into your WordPress `/wp-content/plugins/` directory.
* Go to **WordPress Admin > Plugins** and **Activate** the plugin.

* Copy `edumate-theme/` folder into `/wp-content/themes/`.
